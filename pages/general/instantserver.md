# Deploying an Instant Server

---

## Overview

1. Navigate to the [Deploy Instant Server Page](https://fluidstack.consoledock.com/deploy)

2. Select specifications for server

    - Graphics card model
    - Resource allocation
    - Location
    - Password and Hostname
    
3. Click deploy server and wait for a few minutes
![Deploying Server](/static/deploying_server.png)

4. You will be redirected shortly to the server that was just deployed. If you are not redirected in the next 10 minutes, please contact <support@fluidstack.io>
